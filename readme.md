# BOT TW INFERNO

url: https://infernal-wars.com/

Minimum Requisites:
SO Linux / MacOX,
Node 8+ installed

Installation:

clone this repo:

Set ENV Variables
```
export LOGIN=YOUR_LOGIN_TW_INFERNO && 
export PASSWORD=YOUR_PASSWORD_TW_INFERNO && 
export TW_WORLD=YOUR_TW_WORD
```

Install dependecies 
```
npm install
````

Run BOT: 

```
node runner.js
```
